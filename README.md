# Student_Study_Portal
Student Study Portal is a web-based academic management platform designed to improve students' productivity, learning, and time management. The system integrates essential study tools such as note-taking, homework tracking, and to-do list management in a single dashboard. It also provides quick access to educational resources including YouTube, Wikipedia, and a dictionary for enhanced learning support. Additionally, the portal features book management and various conversion utilities, helping students organize study materials and perform academic tasks efficiently.

## 🚀 Features
- 📝 Create and manage study notes
- 📋 Homework tracking system
- ✅ To-Do List management
- 🎥 Quick access to YouTube educational content
- 🌐 Wikipedia integration for research
- 📚 Dictionary integration
- 🔒 Secure authentication and user management

## 🎯 Objectives
- Improve student productivity
- Simplify academic task management
- Provide easy access to learning resources
- Enhance study planning and organization

## 🛠️ Technologies Used
- **Backend:** Django, Python
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Database:** SQLite
- **Version Control:** Git & GitHub
